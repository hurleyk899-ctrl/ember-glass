# 🌿 Ember Glass

> Your all-in-one cannabis needs app

Ember Glass is a modern cannabis companion app designed to help users discover strains, find local dispensaries, track their preferences, and stay informed — all in one place.

---

## ✨ Features

- 🌱 **Strain Explorer** — Browse and search thousands of cannabis strains with detailed effects, flavors, and terpene profiles
- 📍 **Dispensary Finder** — Locate nearby dispensaries with menus, hours, and reviews
- 📓 **Personal Journal** — Track your experiences, doses, and effects over time
- 💰 **Deals & Promotions** — Never miss a discount at your favorite shops
- 🔔 **Smart Alerts** — Get notified when your favorite products are back in stock

---

## 🛠 Tech Stack

| Layer | Technology |
|-------|-----------|
| Frontend | React Native (Expo) |
| Backend | Node.js / Express |
| Database | PostgreSQL |
| Auth | JWT + OAuth2 |
| Maps | Google Maps API |
| Strain Data | Cannabis APIs |

---

## 🚀 Getting Started

```bash
# Clone the repo
git clone https://github.com/hurleyk899-ctrl/ember-glass.git
cd ember-glass

# Install dependencies
npm install

# Set up environment variables
cp .env.example .env

# Start development server
npm run dev
```
